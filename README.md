# Тестовое в квартирку

# Запуск проекта
```bash
yarn install # Установка пакетов
docker-compose --profile dev up
# Или альтернативно
yarn workspace @app/entrypoint dev
# Сборка
yarn workspace @app/entrypoint build
```
